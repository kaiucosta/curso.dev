## Notes
NVM is a tool for managing multiple versions of Node.js.
NPM is a package manager for Node.js.
Next.js is a React framework for building web applications.
React DOM is a library for rendering React components to the DOM.
React is a JavaScript library for building user interfaces.
RC on file .nvmrc is Run Command to use the node version specified on the file.

## Setup

```bash
# Install nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.4/install.sh | bash

# Install node lts/hydrogen
nvm install lts/hydrogen

# Use node lts/hydrogen
nvm use lts/hydrogen

# Verify node version
node -v
```
